rsay
===

A rusty fork of [gnusay](http://gnusay.sourceforge.net/)

Usage
---

Install via `cargo install rsay` (executables coming soon)

```shell
Usage: rsay [-OPTIONS] [ARG...]

Options:
    -h, --help          Print this help menu
    -W, --width 50      Width of output
```
